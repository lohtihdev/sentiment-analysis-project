⚙️ Pipeline Architecture
Six modular stages from raw text to live prediction.
📂
Data Generation
10,000 synthetic tweets generated with realistic patterns, hashtags, emoji, and retweet structure. Labelled Positive / Negative / Neutral.
🧹
Text Preprocessing
URL removal → HTML stripping → Tokenization (TweetTokenizer) → Stopword removal (keeping negations) → Lemmatization.
🔢
TF-IDF Vectorization
15,000 features, unigrams + bigrams, sublinear TF scaling. Converts text into a sparse numerical matrix for the classifier.
🤖
Logistic Regression
Multi-class classifier with L2 regularization (C=1.0), LBFGS solver. Trained on 8,000 tweets, tested on 2,000.
📈
Evaluation & Visuals
100% accuracy, 5-fold cross-validation. 8 publication-quality charts generated with Matplotlib and Seaborn.
⚡
Live Inference
Saved model (joblib) served via Flask REST API. Real-time predictions with per-class confidence scores.
Built with Python · Scikit-learn · NLTK · Flask · Matplotlib
